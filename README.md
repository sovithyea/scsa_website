# SCSA Swinburne — Official Website

Official website for the **Swinburne Cambodian Student Association (SCSA)** — a space to connect, share culture, and build lasting friendships among Cambodian students at Swinburne University of Technology, Melbourne.

**Live site:** _coming soon_

---

## Tech Stack

- [React](https://reactjs.org/) — UI library
- [React Router](https://reactrouter.com/) — client-side routing
- [Material UI (MUI)](https://mui.com/) — component library & styling

---

## Cuurent Project Structure

```
src/
├── Assets/          # Images and logos
├── components/
│   ├── Header.js    # Top navigation bar (with mobile drawer)
│   ├── Hero.js      # Landing hero section
│   └── Footer.js    # Stay Connected + copyright
└── pages/
    ├── Home.js      # Home page
    ├── About.js     # About Us page
    └── Events.js    # Events page
```

---

## Getting Started

**Clone the repo**
```bash
git clone https://github.com/sovithyea/scsa_website.git
cd scsa_website/scsa_website
```

**Install dependencies**
```bash
npm install
```

**Run locally**
```bash
npm start
```

Opens at `http://localhost:3000`
